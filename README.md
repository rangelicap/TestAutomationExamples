# TestAutomationExamples
Prerequisites: Visual Studio (Community should work fine)
To ensure the test results file is generated, select/configure the .runsettings file in VS. In the Test > Configure Run Settings select 'Auto Detect runsettings Files'. Or alternatively select the option 'Select Solution wide runsettings File' and select the testUI.runsettings file located in the TestAutomationExamples.Specs folder

Steps to run automation:
1. In Visual Studio under the Build menu select Clean Solution then Rebuild Solution
2. Once rebuild is successful the tests should appear in the Test Explorer window. If Test Explorer is not displayed, go to Test > Test Explorer
3. Select a test and click the Run button in the Test Explorer toolbar or right-click the test and select 'Run'
